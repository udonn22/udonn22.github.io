カイ二乗分布ってどんな分布?

博士、今度はカイ二乗分布っていうのを聞いたんだけど、それは何？

カイ二乗分布は統計学でよく使われる分布の一つだよ。これは、独立した標準正規分布に従う複数の変数の二乗和が従う分布を指すんだ。

標準正規分布はこの間教えてもらったけど、その二乗和ってどういうこと？

もちろんだよ。標準正規分布ってのは、平均が0で、標準偏差が1の正規分布のことを言うんだったよね。

そして、その分布に従うランダムな変数をいくつか取って、それぞれを二乗して足し合わせたものが、カイ二乗分布に従うってわけだ。

式で表すとこうだね。

\[ X_1, X_2, …, X_\nu \sim N\left(0, 1\right) \qquad i.i.d. \] のとき、 \[ Y = X_1^2+X_2^2+…+X_\nu^2 \] の\(Y\)は自由度\(\nu\)のカイ二乗分布に従う。 ただし、\(i.i.d.\)とは、独立同分布(independent and identically distributed)を表す。

自由度って何？

簡単に言うと、自由度とは分析に使われる独立した情報のピースの数のことだよ。例えば、2つの変数の二乗和を考えた場合、そのカイ二乗分布は2の自由度を持つと言うわけだ。

2つの変数は例えば、“薬を飲んだかどうか”と、“病気の症状の重さ”を数値にしたものがあるよ。自由度は、分布の形を決定づける重要な要素なんだ。

どんなときにカイ二乗分布を使う？

なるほどね。でも、どんなときにカイ二乗分布を使うの？

カイ二乗分布はデータの分散や予想外の偶然の一致をテストするのに非常に役立つんだ。

例えば、ある薬が効果があるかどうかを調べたいとき、実際のデータと期待されるデータがどれだけ違うかをカイ二乗分布を使って評価することができるんだ。

おお、それは面白そうだね！

とても面白いよ! そして、カイ二乗分布には自由度というパラメータがあってね。

へえ、なんだか難しいけど、カイ二乗分布って大事なんだね！

まさにその通り。カイ二乗分布は統計的検定において非常に重要で、研究者がデータが持つ情報を最大限に活用するのを助けるんだよ。だから、科学や社会科学の多くの分野で使われているんだ。

カイ二乗分布ってどんな形?

カイ二乗分布ってどんな形になるの?正規分布からできるから、ベル状になるの?

いいや、カイ二乗分布は二乗した値を足し合わせたものだから、左右対称にならないんだ。実際にプログラミングで見てみよう。

【クリックでPythonコードを表示】 
import numpy as np
import matplotlib.pyplot as plt
import scipy.stats as stats
import japanize_matplotlib

# 自由度のリスト
degrees_of_freedom = [2,3,4, 5, 10]

# Yの値
Y = np.linspace(0, 20, 1000)

# Plot each chi-squared distribution
plt.figure(figsize=(6, 4))
for df in degrees_of_freedom:
    plt.plot(Y, stats.chi2.pdf(Y, df), label=f'自由度:{df}')

plt.title('カイ二乗分布の確率密度関数')
plt.xlabel('Y')
plt.ylabel('確率密度')
plt.legend()
plt.grid()
plt.show()

ほんとだ! 左右が対象じゃないね! そして自由度が増えるにつれて、分布が右の方に移っていくんだね!
