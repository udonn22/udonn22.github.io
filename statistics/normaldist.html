正規分布: 自然界と社会でのパターン

正規分布ってどんな分布?

博士、正規分布ってよく聞くけど、それはどういう意味なの?

正規分布は、自然界や人間の行動においてよく見られるデータの分布のことだよ。例えば、人々の身長やテストの点数がこの分布に従っていることが多いんだ。

この分布は、下の図みたいにベルの形をしていて、データの多くが平均値の周りに集まり、極端に高い値や低い値は少なくなる特徴があるんだ。

【クリックでPythonコードを表示】 
# !pip install japanize-matplotlib
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm
import japanize_matplotlib

# 正規分布のパラメータ
mu = 0  # 平均
sigma = 1  # 標準偏差

# 正規分布に従うデータの生成
x = np.linspace(mu - 3*sigma, mu + 3*sigma, 1000)
y = norm.pdf(x, mu, sigma)

# 正規分布のプロット
plt.figure(figsize=(6, 4))
plt.plot(x, y, label='正規分布')
plt.title('正規分布 (μ=0, σ=1)')
plt.xlabel('X')
plt.ylabel('確率密度')
plt.legend()
plt.grid(True)
plt.show()
ちなみに確率分布の式はこうだよ。

\[ f(x|\mu,\sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \] ただし、\(x\)は確率変数、\(\mu\)は平均（期待値）、\(\sigma^2\)は分散、\(\sigma\)は標準偏差です。

確率変数\(X\)が正規分布に従うときは、以下のようにして書くことが多いから、覚えておくといいよ。

\[ X \sim N\left(\mu, \sigma^2\right) \]

単純にかけてわかりやすいね!

正規分布はどんな時に使われる?

どうしてベル状の形になるの?

いい質問だね。実は、結果に多くの独立した因子の影響うけると、その結果は自然と正規分布に従うことが多いんだ。たくさんの小さなランダムな影響が合わさって、最終的にはこのような形に落ち着くんだよ。

なるほど、いろんな要素が組み合わさって、平均的な値が一番多くなるわけだね。正規分布はどんな時に役立つの?

正規分布は、予測や意思決定に非常に役立つんだ。例えば、学校のテストで平均点や合格基準を設定する時や、製品の品質管理、さらには科学研究で観測データの解析にも使われるよ。

わあ、正規分布って本当にいろんなところで使われているんだね! 次から見るデータが正規分布に従っているかどうか、気にしてみるよ!

標準正規分布: データの標準化の理解

博士!標準正規分布って何ですか?

標準正規分布は、統計学で非常に重要な分布だよ。データの分布を標準化し、比較や解析を容易にするためのツールなんだ。

具体的には、平均が0、分散が1の正規分布のことを指すんだ。この標準化を行うことで、異なるデータセットを同じ尺度で比較することができるようになるんだよ。

ちなみに標準正規分布の式はこうだよ。

\[ f(x|0,1) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}} \]

とってもシンプルな形になったね! 標準化っていうのはデータを特定の形に変換することなんだね!

データのばらつきと平均をそろえる方法とは?

どうやって標準正規分布に変換するの?

いい質問だね! データを標準正規分布に変換するには、「標準化」という手順を使うんだ。各データ点から平均を引いて、それを標準偏差で割るんだ。これによって、新しいデータセットの平均は0、分散は1になるよ。

ちなみに式はこうだよ。

\[ z = \frac{x - \mu}{\sigma} \] ただし、\(x\)は確率変数、\(\mu\)は平均（期待値）、\(\sigma\)は標準偏差です。

へえ、そうするとどんなデータも同じ基準で考えられるようになるんだね! それで、標準正規分布が使われるのはどんな時なの?

標準正規分布は、科学研究から金融分析まで、幅広い分野で利用されているよ。例えば、テストの成績の比較、品質管理、リスク管理などに使われるんだ。標準化されたデータを使うことで、異なる条件や背景を持つデータ間で公平な比較が可能になるんだ。

わあ、標準正規分布って本当に便利なんだね! これからデータを見る時は、標準化のことも考えながら見てみるよ!

標準化をした正規分布の例

最後に正規分布を標準化したときの分布を比べてみるね

わあ、標準正規分布って本当に便利なんだね! これからデータを見る時は、標準化のことも考えながら見てみるよ!

【クリックでPythonコードを表示】 
# 平均が1、分散が2の分布のデータを標準化する

# 元の分布のパラメータ
mu_original = 1  # 平均
sigma_original = np.sqrt(2)  # 分散が2のため、標準偏差はsqrt(2)

# 正規分布に従うデータの生成（元の分布）
x_original = np.linspace(mu_original - 10*sigma_original, mu_original + 10*sigma_original, 1000)
y_original = norm.pdf(x_original, mu_original, sigma_original)

# データの標準化
x_standardized = (x_original - mu_original) / sigma_original
y_standardized = norm.pdf(x_standardized, 0, 1)

# 元の分布と標準化された分布のプロット
plt.figure(figsize=(6, 3))

# 元の分布のプロット
plt.plot(x_original, y_original, label='標準化前(μ=1, σ²=2)', color='blue')

# 標準化された分布のプロット
plt.plot(x_standardized, y_standardized, label='標準化後(μ=0, σ²=1)', linestyle='--', color='red')

plt.title('標準化前後の正規分布の比較')
plt.xlabel('X')
plt.xlim(-5, 5)
plt.ylabel('確率密度')
plt.legend()
plt.grid(True)
plt.show()

平均が0になって、分散も変わっているね! 標準化って簡単にできるんだね!
