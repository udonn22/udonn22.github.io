t分布について

t分布ってどんな分布?

博士、今度はt分布っていうのを聞いたんだけど、それは何？

t分布もまた統計学で非常に重要な分布だよ。t分布は、サンプルサイズが小さいときに、平均値の信頼区間を推定するのに使われるんだ。

信頼区間って何？

信頼区間とは、あるパラメーター（例えば平均値）が存在すると推定される範囲のことだよ。言い換えると、僕たちが求めた推定値にどれだけ「信頼」を置いていいのかを数値で表す方法なんだ。

そうなんだ。推定値を信頼できるかどうかを評価できるって大事そうだね! でも、なんで小さいサンプルサイズでt分布を使うの？正規分布じゃダメなの?

いい質問だね。小さいサンプルサイズの場合、データに外れ値が入ったりしたら、正規分布だととても大きな影響を受けてしまうんだ。

でもt分布は、サンプルサイズが小さい場合にも、より正確な推定を可能にするんだよ。これは正規分布に比べて、t分布は裾が重くなっているから、外れ値の影響を受けにくいからなんだ。

裾が重いってどういうこと？

裾が重いとは、グラフの端っこ、つまり値が大きい部分や小さい部分が、より広がっているという意味だよ。これによって、データの変動が大きい場合にも、その影響を考慮に入れることができるんだ。

グラフで確認してみよう

t分布の分布形状

グラフを見る前に、以下のt分布の定義を確認しよう。

\[U \sim N(0,1),\] \[V_m \sim \chi^2(m), \] \[U \text{と} V_m \text{は独立とするとき、}\] \[T = \frac{U}{\sqrt{V_m/m}} \] \[\text{は自由度$m$のt分布$t(m)$に従う.}\]

t分布にも自由度っていうのがあるんだね。これもカイ二乗分布と同じものなの?

そうだよ。簡単に言うと、自由度とは分析に使われる独立した情報のピースの数のことだよ。

平均値の信頼区間を考えるときは、サンプルサイズから1を引いた値になるよ。

じゃあ自由度を変えたt分布のグラフってどうなるの?

実際にグラフを見てみよう!

# Re-import necessary libraries after code execution state reset
import numpy as np
import matplotlib.pyplot as plt
import scipy.stats as stats
import japanize_matplotlib

# t分布の自由度
degrees_of_freedom_t = [1, 3, 10, 100]

# X values for our plot
x_t = np.linspace(-5, 5, 1000)

# Plot each t-distribution
plt.figure(figsize=(6, 4))
for df in degrees_of_freedom_t:
    plt.plot(x_t, stats.t.pdf(x_t, df), label=f'自由度:{df}')

plt.title('自由度ごとのt分布の確率分布')
plt.xlabel('t')
plt.ylabel('確率分布')
plt.legend()
plt.grid()
plt.show()

自由度が低いときのt分布はグラフの両端でも値が大きいね!これが”裾が広い”ってことだね!

そうなんだよ! 実はサンプルサイズと自由度をどんどん大きくしていくとt分布は正規分布に近づくんだ。逆に、サンプルの数が少ないときにt分布は外れ値の影響を減らせるんだよ!

t分布の利点

じゃあ、t分布を使うと何がいいの？具体的に知りたいな。

t分布を使う最大の利点は、小さいサンプルサイズでのデータ分析がより正確になることだよ。

例えば、あるクラスでのテストの平均点を計算するとき、そのクラスの生徒数が少ない場合、t分布を用いて平均点の信頼区間を推定すると、その結果はより信頼できるものになるんだ。

なるほどね！だから、小さいサンプルで研究するときは、t分布がとても役に立つんだね。

その通り！t分布は、特に小規模な研究や前もってサンプルサイズを大きくできない場合に非常に有効なツールなんだ。これにより、限られたデータからでも、より正確な統計的推論を行うことができるようになるんだよ。

t分布と正規分布への外れ値の影響

外れ値の影響について簡単に見てみよう。平均0、分散1の正規分布からランダムな数字を9つ作って、そこに外れ値として20を追加した、合計10個のサンプルの場合を考えてみるね。

このサンプルから正規分布とt分布の分布を作成したものが以下になるんだ。

【クリックでPythonコードを表示】 
import numpy as np
import matplotlib.pyplot as plt
import scipy.stats as stats

# データセットを生成（9つの正常値と1つの異常値）
np.random.seed(0)
normal_values = np.random.normal(loc=0, scale=1, size=9)  # 正常値
outlier = np.array([20])  # 異常値
values_with_outlier = np.append(normal_values, outlier)  # 正常値と異常値を結合

# 平均と標準偏差を計算
mean_outlier = np.mean(values_with_outlier)
std_outlier = np.std(values_with_outlier, ddof=1)

# 自由度（データ数-1）
df = len(values_with_outlier) - 1

# 正規分布の範囲を定義
x_outlier = np.linspace(mean_outlier - 4*std_outlier, mean_outlier + 4*std_outlier, 100)

# t分布の範囲を定義
x_t = np.linspace(-4, 4, 100)

# グラフを描画
plt.figure(figsize=(6, 4))

# 異常値を含む正規分布をプロット
plt.plot(x_outlier, stats.norm.pdf(x_outlier, mean_outlier, std_outlier), label='異常値を含む正規分布', color='blue')

# t分布をプロット
plt.plot(x_t, stats.t.pdf(x_t, df, 0, 1), label='t分布', color='orange')

# サンプルの位置に点をプロット
plt.scatter(values_with_outlier, np.zeros_like(values_with_outlier), color='green', zorder=5, marker='x', label='サンプルの位置')


plt.title('異常値を含む正規分布とt分布の比較')
plt.legend()
plt.grid(True)  # グリッドを表示
plt.show()

正規分布は外れ値に引っ張られて中心が0から離れているね。分布も広がっているし、これだと平均値を精度よく求められないね。

そうなんだ。サンプルサイズが小さいときは特に正規分布はとても外れ値の影響を受けやすいんだ。でもt分派は外れ値の影響がずっと小さくてすむんだ。
