中心極限定理

中心極限定理ってなに?

博士! 正規分布については分かったけど、中心極限定理って何？

中心極限定理は非常に重要な概念だよ。これは、大きなサイズの標本を取ると、その標本の平均は、たとえ元のデータがどんな分布をしていたとしても、正規分布に近づくという定理なんだ。

それはどういうこと？

例えで説明するよ。君がランダムに100個の小石を拾って、その小石の重さの平均値を記録する。この拾って記録するのを何百回も繰り返すことを想像してみてね。

それでどうなるの？

もし君がその平均値をたくさん集めて、その分布をグラフにすると、驚くことに、それらの平均値は正規分布、つまりベル曲線の形になるんだ。元の小石の重さがどのような分布をしていたとしても、平均値の分布は正規分布に近づく。これが中心極限定理の美しい点だよ。

えーすごいね!なんでそうなるの？

これは統計学の一部で、独立したランダムな変数の和（この場合は小石の重さ）は、サンプルサイズが大きくなるにつれて、その平均値が正規分布に近づくという性質があるからだよ。

だから、君が小石の重さの平均を取る行為を繰り返すことで、その平均値は特定のパターン、つまり正規分布に従うようになるんだ。

へえ、だから何度も試すと、平均値はみんな一緒の形になるんだね！

そういうこと！そして、この原理は科学や工学、経済学など、あらゆる分野でデータを分析する際に役立つんだ。中心極限定理のおかげで、さまざまなデータセットから有意義な結論を導き出せるんだよ。

中心極限定理をPythonで確認

ここでは、サイコロの出目の例を通じて、中心極限定理を理解してみよう。

1.サイコロの出目の生成

中心極限定理に行く前に、今回の確率分布について理解しよう。

10000回、1から6が同じ確率で出るサイコロの出目を生成してみるよ。このような、確率の分布が一定の場合の分布を一様分布と言うよ。

【クリックでPythonコードを表示】 
import numpy as np
import matplotlib.pyplot as plt
import japanize_matplotlib

# サンプル数(サイコロを何回振るか)
n_trials = 10000

# 一様分布からのサンプリング
samples = np.random.randint(1, 7, n_trials)

# ヒストグラムで分布を可視化
plt.figure(figsize=(6, 3))
plt.hist(samples, bins=np.arange(0.5, 7.5), density=True, alpha=0.7)
plt.title(f'1から6までの一様分布で{n_trials}回試行した結果')
plt.xlabel('確率変数X')
plt.ylabel('頻度の割合')
plt.grid(True)
plt.show()

1から6までの目が同じくらい出てることがわかるね!

2.中心極限定理の検証

次に中心極限定理を検証するよ。中心極限定理によれば、\(n\)回サイコロを投げた時の平均の分布は、\(n\)が大きくなるにつれて正規分布に近づくはず。

ここでは、サイコロを2回投げた平均\(\bar{X}_i\)を求める試行を1セットとして、10000セット分の試行をした場合の平均の分布を確認するよ。

「サイコロを2回投げてその平均\(\bar{X}_i\)を記録する」のを10000回繰り返すんだね!
そうだよ。1セットのなかで投げるサイコロの数を2から5、10、100と多くすると、\(\bar{X}_i\)の分布は正規分布に近づくはずだよ。

【クリックでPythonコードを表示】 
# 図を作成する関数を作成
# n_values: 一度の試行でサイコロを振る回数
# n_trials: 試行回数
def plot_histogram(n_values, n_trials):
    # 平均の計算
    means = np.mean(np.random.randint(1, 7, (n_trials, n_values)), axis=1)

    # ヒストグラムで分布を可視化
    plt.figure(figsize=(6, 3))
    plt.hist(means, bins=np.arange(1-1/n_values,6+1/n_values*2,1/n_values), density=True, alpha=0.7)
    plt.title(f'{n_values}回サイコロを投げて平均を算出する試行を{n_trials}回繰り返した結果')
    plt.xlabel(r'各試行での確率変数の平均値$\bar{X}$')
    plt.ylabel('頻度の割合')
    plt.xlim([1-1/n_values, 6+1/n_values*2])
    plt.grid(True)
    plt.show()
【クリックでPythonコードを表示】 
plot_histogram(n_values=2, n_trials=10000)
サイコロ2回だと、ピラミッドみたいな形だね!
【クリックでPythonコードを表示】 
plot_histogram(n_values=5, n_trials=10000)
サイコロ5回になると、かなりベルの形になるね!
【クリックでPythonコードを表示】 
plot_histogram(n_values=100, n_trials=10000)
サイコロ100回もベルの形だけど、なんだか分布が真ん中に集まってない?
いいところに気づいたね! これは中心極限定理の正規分布の形から説明できるよ。

中心極限定理の定義

ちょっと難しいけど中心極限定理の定義はこうだよ。

ある母集団から無作為に抽出されたサイズ\(n\)の標本の平均\(\bar{X}\)は、\(n\) が十分に大きい場合に以下の分布に近づく \[ f \left(x \left|\mu,\frac{\sigma^2}{n}\right) \right. = \frac{1}{\sqrt{2\pi (\sigma^2/n)}} e^{-\frac{(x-\mu)^2}{2(\sigma^2/n)}} \] ただし、\(μ\)は母集団の平均、\(σ^2\)は母集団の分散である。 これは、元の母集団の分布が正規分布でなくても成立する。

\(n\)が十分に大きいときは、正規分布を\(N\)で表すと、nが増えた時に\(\bar{X}\)は以下の式で近似的に考えることができよ。

\[ \bar{X} \sim N\left(\mu,\frac{\sigma^2}{n}\right) \]

あ! 分散が\(\sigma^2/n\)になっている! 平均をとるサンプルの数\(n\)が大きくなるとそれに反比例して分散が減るんだね!
とういうこと!だからさっきの例だと、サイコロを投げる数が100回のときは分布が中央に集まっていたんだよ!
